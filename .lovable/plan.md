# Добавление кнопки MAX рядом с Telegram

## Что меняется
Везде, где сейчас есть ссылка/кнопка на Telegram (`https://t.me/Chargeback_2022`), добавляется вторая такая же кнопка на MAX:
`https://max.ru/u/f9LHodD0cOJUE7hx1DQDtMKLWkJV-OtVn834RkVcrxac9K-YV6BfVKzEpj8`

## Где появятся кнопки MAX
1. **Hero (главный экран)** — рядом с «Бесплатная консультация (Telegram)» добавляется вторая кнопка «Написать в MAX».
2. **Navbar (шапка)** — рядом с кнопкой «Консультация» (Telegram) добавляется вторая компактная кнопка «MAX» (десктоп + мобильное меню).
3. **Блок «Контакты»** — под строкой Telegram добавляется ещё одна строка контакта: «MAX» со ссылкой.

## Визуальное оформление
- Кнопки MAX используют тот же стиль, что и кнопки Telegram (зелёный CTA для основных, контурный для второстепенных), чтобы сохранить единый дизайн.
